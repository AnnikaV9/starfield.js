starfield.js

An interactive star field background effect with html canvas. Uses no external libraries.

How do I use this?

Grab the code and read the guide to get started.

Inspiration

Inspired by Barney Code's Star Field Hyperdrive Light Speed Effect, which was built on p5.js.

This is a more configurable, production ready implementation that uses canvas directly.

Can I use this anywhere?

Pretty much, it's licensed under the MIT License so go crazy with it.

On mobile?

Hover or drag won't work.

Tap the origin instead to
accelerate the stars.
